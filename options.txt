Usage: ascii.rb [name|number] or ascii.rb [options]
    (note that you can always only handle one project)
    -p, --project NAME               Use this project (overrides others)
    -n, --new NAME                   Create new Project
    -e, --edit [NAME]                Edit Project
    -i, --invoice [NAME]             Create invoice from project
    -o, --offer [NAME]               Create offer from project
    -l, --list                       List all projects
    -s, --sum [NAME]                 Sum up project sum
    -d, --dump [NAME]                Dump raw project data
    -f, --file PATH                  Manualy specify .yml file
        --close NAME                 Close project 
        --reopen NAME                Move closed project back to working dir (Combine with -a)
    -a, --archive [year]             List archived projects (default is current year)
    -k, --keep-log                   Do not delete latex log and aux files
    -v, --verbose                    Be verbose
    -V, --veryverbose                Be very verbose
    -h, --help                       Display this screen
